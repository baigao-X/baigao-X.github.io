# 组件

## 分布式存储
- - **Apache Hadoop (HDFS):** 作为大数据领域的基石，HDFS（Hadoop Distributed File System）

## NoSQL数据库
**Redis:** 一个基于内存的高性能键值存储，通常用作缓存、消息队列和会话存储，能够显著提升应用性能

## NewSQL数据库
兼具传统关系型数据库的ACID特性和NoSQL数据库的扩展性
## 消息队列

- **Apache Kafka:** 一个高吞吐量、分布式的发布订阅消息系统，广泛应用于实时数据管道、流处理和日志聚合等场景。
- **RabbitMQ:** 一个实现了AMQP（高级消息队列协议）的开源消息代理软件，以其可靠性、灵活性和丰富的功能而著称。
- **RocketMQ:** 由阿里巴巴开源的分布式消息中间件，具有高吞吐量、高可用性和可扩展性等特点，在中国互联网公司中得到了广泛应用

## 服务调度

- **Apache ZooKeeper:** 一个为分布式应用提供一致性服务的软件，常用于配置管理、命名服务、分布式锁和集群管理等场景
- **Dubbo:** 一个高性能的Java RPC（远程过程调用）框架，提供了服务注册、发现、负载均衡和容错等核心功能，是构建微服务架构的重要选择

## 监控运维

- **Zabbix:** 一款企业级的开源监控解决方案，能够监控网络参数、服务器健康状况和应用程序等。
- **OpenFalcon:** 由小米开源的企业级监控系统，提供了完善的监控指标和灵活的告警机制，在国内互联网公司中颇受欢迎。
- **CAT**: 由大众点评开源的实时应用监控平台，提供分钟级的监控、告警和应用分析，帮助开发者快速定位线上问题
