# 封装总览<no value>

# 1. MP4\

主要三个box
ftyp 存放编码格式、标准等信息
moov 用于存放元数据，mdat数据块中的映射关系
mdat:音视频信息，以来moov才能推算没帧播放时间。
因此在线播放的MP4需要把moov放在mdat前面


短视频为主，视频文件不大，可以考虑使用，应为支持格式方便。
但是文件较大，或时长过长，长时间在线播放可能会播放中断（因为前面的ftyp和moov数据丢失，没法继续播放），CDN加速效果不佳
# 2. FLV

FLV文件每个音视频数据块都有时间戳标识，所以首帧播放，未加载跳转表现优秀
如果在线播放FLV视频文件 出现未加载跳转错误，大概率是没有建立关键帧索引
大文件、长时间播放比MP4稳定，但是CND加速效果也不好。
