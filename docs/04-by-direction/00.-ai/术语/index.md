### NLP 自然语言处理

### LLM (Large Language Model) 大语言模型
LLM是NLP的一个子集，是NLP领域的技术革命
一种基于深度学习具有大规模参数的预训练语言模型
大语言模型专注于单一文本模态
### 多模态模型

同时处理多种模态数据（如文本、图像、音频、视频）

### Transformer架构
LLM 普遍采用该架构，依赖自注意力机制（Self-Attention）处理长距离文本依赖关系，提升对上下文的理解能力


### Agent
定义为以LLM为大脑驱动，具有自主理解、规划、记忆和工具能力的系统

### MCP (Model Context Protocol) 模型上下文协议
由 Anthropic 协议推出的开放协议，核心目标是通过统一的通讯协议让AI模型能够访问外部资源
