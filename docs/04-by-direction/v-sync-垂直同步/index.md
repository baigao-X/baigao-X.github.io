# V Sync 垂直同步<no value>


垂直同步，实际应该理解为场同步，是显卡提供的一个功能，用于将游戏帧率和显示器刷新率进行同步

- 开启后: 显卡会等待显示器本轮刷新完成。在此期间，不会向显示器传输任何数据。
实际就是设置Present操作会等待垂直重扫完成后，完成front buffer的切换。（若检测到垂直同步完毕，则马上flip调换frontbuffer和backbuffer）。

当FPS高于显示刷新率时: 会出现以下问题：
1. 比如实际显示器刷新率只有60HZ, 画面有120FPS，那么实际只有两帧画面才会更新一次到显示器，那么就有一帧渲染实际上是被浪费的。
2. 当FPS特别高，有可能显示器还未刷新完当前帧，就收到了下一帧新数据，就可能在刷新过程中同时显示两个不同帧的画面，会出现画面撕裂。
因此打开V-Sync可以避免以上现象


当FPS低于刷新率时，打开了V-Sync可能出现以下问题：
- 可能出现显示器完成本轮刷新，显卡还没渲染好下一帧画面：比如显示器刷新率为60HZ，每次刷新间隔为16ms，当前画面为50FPS，即每20ms渲染一帧。那么第16ms，画面没有渲染帧，当前同步无效，第20ms渲染帧数据放置到帧缓冲区，直到下一个显示周期(第32ms)才被刷新。此时花了32ms才出一帧画面，实际FPS为30。
渲染间隔ms: 0->20->40->60->80
显示间隔ms: 0->16->32->48->64->80
  如此反复，直到第80ms，渲染帧赶上了刷新，FPS暂时提高到了50。 因此FPS总会不是平滑渐变的，形成卡顿。
关闭V-Sync可以避免以上现象。
