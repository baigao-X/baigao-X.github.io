客户端#

使用预授权#

  • 服务端：
    • headscale preauthkeys create –user <用户名> –reusable –expiration 7d # 可重复使用7天有效期
    • headscale preauthkeys create –user<用户名> –ephemeral # 临时密钥
  • 客户端：
    • tailscale up –login-server https://headscale.emptyroom.top –authkey 78eee0541be7e238d1cd037e48802eedd8b65ed26da8a839 –hostname=my-laptop

交互式授权#

  • 客户端
    • tailscale up –login-server https://headscale.emptyroom.top
  • 服务端
    • headscale nodes register –user USERNAME –key rqemD8lOBrYA6zIjNdzvdyaz-
