# Boot<no value>


# 1. ROM BIOS

会将引导扇区 bootsect加载到0x7C00，并从该地址开始运行

# 2. bootsect.S


	1.将自己搬移到0x90000地址
	2.将磁盘第二个扇区开始的4个扇区包含的setup模块代码加载到0x90200地址
	3.使用BIOS终端0x13获取磁盘当前引导盘参数
	 4.在屏幕中显示“Loading system”
	 5.将setup内存搬移到0x10000
	 6.确定根文件系统的设备号（root_dev,即引导块的508地址处）,设备号=(主设备号<<8)+次设备号
	（主设备号： 1-内存,2-磁盘,3-硬盘,4-ttyx,5-tty,6-并行口,7-非命名管道）
		0x300 /dev/hd0 - 代表整个第 1 个硬盘； 
		0x301 - /dev/hd1 - 第 1 个盘的第 1 个分区
		0x304 - /dev/hd4 - 第 1 个盘的第 4 个分区
		0x305 - /dev/hd5 - 代表整个第 2 个硬盘 
		0x306 - /dev/hd6 - 第 2 个盘的第 1 个分区

运行时所在物理地址： 0x90000-0x90200
