## 2.1. 寄存器
### 2.1.1. 内存管理寄存器

 -有4个基地址寄存器 GDTR、LDTR、IDTR、TR
 ![[Pasted image 20240709090159.png]]

#### 2.1.1.1. GDTR (全局描述符表GDT寄存器)

	GDTR 指向GDT表

- 指令： LGDT、SGDT

#### 2.1.1.2. IDTR (中断描述符表寄存器 )
	IDTR 指向 IDT表
- 指令： LIDT 、 SIDT 
#### 2.1.1.3. TR (任务寄存器)
	TR 指向当前任务的TSS
- 指令： LTR： 初始化任务寄存器
## 2.2. 段

### 2.2.1. 寻址

通过**段选择符**找到对应的**段描述符**，通过**段描述符**得到段基地址，加上偏移地址得到线性地址
![[Pasted image 20240709092132.png]]

### 2.2.2. 段选择符

- 描述符索引
- TI： 0-表示使用GDT，1-表示使用LDT
- RPL：请求特权级
![[Pasted image 20240709092307.png]]
### 2.2.3. 段描述表

虚拟地址空间被分割成大小相等的两半，一半由GDT来映射为线性地址，一半由LDT来映射线性地址

![[Pasted image 20240709092844.png]]

#### 2.2.3.1. 全局描述符表 GDT 

系统中所有任务共用
第一个描述符为空，不使用
每一项指向共用的内核数据段、内核代码段、以及各个任务的LDT、TSS

#### 2.2.3.2. 局部描述符表 LDT
任务独占，实现虚拟地址空间隔离。每个任务有一个。

#### 2.2.3.3. 段描述符
 段描述符表中的一项, 占64bytes（8字节），主要分为三个部分 **段基地址、段限长、段属性**

![[Pasted image 20240709092637.png]]
