# Ngtcp2<no value>


ngtcp2 库实现了三种主要的拥塞控制算法：

1. **RENO** - A simple algorithm using additive increase/multiplicative decrease (AIMD)  
    **RENO** - 一种使用加法增加/乘法减少 （AIMD） 的简单算法
2. **CUBIC** - An enhanced algorithm with a cubic growth function  
    **CUBIC** - 具有三次增长函数的增强算法
3. **BBR** - A model-based algorithm using bandwidth and RTT measurements  
    **BBR** - 使用带宽和 RTT 测量的基于模型的算法
