# G.711<no value>

G.711是一种由国际电信联盟（ITU-T）制定的音频编码方式，又称为ITU-T G.711。
采用对数PCM抽样标准
采样率为8KHz
是主流的波形声音编码

---

## 0.1. G.711A
	 采用 µ-law algorithm 压缩算法 （又称often u-law, ulaw, mu-law)， 主要运用于北美和日本
    特别设计来方便计算机处理
	将13bit的PCM数据编码成8bit数据

## 0.2. G.711U
	 采用 a-law algorithm 压缩算法 主要运用于欧洲和世界其他地区
	将14bit的PCM数据编码成8bit数据
